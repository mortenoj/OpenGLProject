# OpenGLProject

An unnamed OpenGL project for learning rendering in C++.

## Setup

### OSX

- Open xcodeproj file in xcode.
- Check that libraries linked properly (i.e SOIL2, glm, glew, glf3)

### Windows

- Open visual studio project file in visual studio
- Check that libraries linked properly (i.e SOIL2, glm, glew, glf3)
